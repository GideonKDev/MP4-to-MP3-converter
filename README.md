# MP4 to MP3 Converter Pro

A professional desktop application for converting MP4 video files to MP3 audio format with a beautiful GUI interface.

![Screenshot](https://screenshot.png)

## Features

* **Drag & Drop** interface for easy file selection
* **Batch conversion** - process multiple files at once
* **Modern GUI** built with PyQt6
* **Real-time progress** tracking
* **Customizable settings** (bitrate, format, normalization)
* **Metadata preservation** from source files
* **Cover art extraction** and embedding
* **Dark/Light theme** support
* **Conversion statistics** and history

## Installation

### Option 1: Download Pre-built EXE (Windows)

1.  Go to **Releases**
2.  Download `MP4toMP3_v1.0.0.zip`
3.  Extract and run `MP4toMP3.exe`

### Option 2: Build from Source

**Prerequisites**

* Python 3.8 or higher
* FFmpeg installed on your system

**Step 1: Clone the repository**

```bash
git clone [https://github.com/yourusername/mp4tomp3-converter.git](https://github.com/yourusername/mp4tomp3-converter.git)
cd mp4tomp3-converter
```
### step 2: Install dependencies

```bash
pip install -r requirements.txt
```
### Step 3: Run the application

```bash
python setup.py install
```
### Step 4: Launch the application

```bash
python mp4tomp3.py
```
## Usage
1.  Open the application
2.  Drag and drop MP4 files into the interface
3.  Configure settings if needed
4.  Click "Convert" to start the conversion process
5.  Monitor progress and access converted MP3 files in the output directory
